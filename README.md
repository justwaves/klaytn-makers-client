## Install flow

#### Clone Repository

```
git clone https://github.com/justwaves/klaytn-makers-client
cd klaytn-makers-client
```

#### Install Truffle

```
npm install -g truffle
```

#### Install Dependencies

```
yarn install
```

#### Start Repository

```
yarn start
```
